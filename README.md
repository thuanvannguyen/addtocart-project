# Cấu trúc thư mục Redux

Dự án này sử dụng cấu trúc thư mục dựa trên tính năng cho Redux, giúp quản lý và mở rộng dễ dàng hơn. Dưới đây là mô tả chi tiết về cấu trúc thư mục của Redux trong dự án.

## Cấu trúc thư mục

```plaintext
src
└───redux
    └───features
        └───details
        │       detailsSlice.js
        └───navbar
        │       navbarSlice.js
        └───products
                productsSlice.js
```
